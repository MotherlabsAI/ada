# ROOT.000 · Ada Surface Design World Model

> ∴ inference · L5 · C0 · area **ROOT** · from `SEED (root intent)`

## Summary
The bounded design context for Ada's own terminal surface — not everything about TUIs, only what an executor needs to build/refactor THIS shell: identity, palette, layout, motion, nav, flow, state, a11y, voice, and honest residue.

## Why it matters
The map the surface is built inside. Without it, the TUI is restyled from taste-of-the-day instead of a governed model.

## Failure if missing
Each screen gets designed in isolation; the shell drifts into confetti and inconsistency.
