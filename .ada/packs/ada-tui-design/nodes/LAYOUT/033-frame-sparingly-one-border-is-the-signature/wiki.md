# LAYOUT.033 · Frame sparingly — one border is the signature

> ∴ inference · L2 · C2 · area **LAYOUT** · from `docs/SURFACE-DESIGN.md`

## Summary
One rounded border (the Welcome box / `?` overlay) is a signature; bordering every pane is noise. Inside the workbench, separate regions with the StatusBar title and a dim hint line — no nested boxes, no second border style.

## Why it matters
Borders are expensive attention; spending them once makes that one frame meaningful.

## Failure if missing
Every pane gets a box; the screen becomes a grid of cells and the signature frame loses all weight.
