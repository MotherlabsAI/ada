# FLOW.061 · Compile journey — a sentence becomes a pack

> ∴ inference · L4 · C1 · area **FLOW** · from `docs/SURFACE-DESIGN.md`

## Summary
The core trust moment: one sentence of intent → the 9-stage heartbeat (visible, determinate) → a finished pack the user can open. Input is usable immediately (no blocking intro >3s); completed stages scroll into <Static>; the end state hands off to the graph, not a dead 'done'.

## Why it matters
This is A8 made tangible — the one journey that must feel like a real compiler doing real work, because it's where the product earns its price.

## Failure if missing
The compile feels like a black-box spinner; the user can't distinguish progress from a hang and never trusts the output.
