# FLOW.060 · Welcome home — recognition over recall

> ∵ source · L4 · C2 · area **FLOW** · from `src/tui/ink/Welcome.ts`

## Summary
The home screen SHOWS the actions (Compile / Open / Interview / Browse / Settings) with an arrow-nav menu, a sidebar that narrates the focused item, and a 'your projects' panel of packs on disk with node/κ/Ω counts — not a memorized command list. Key hints are the 3–5 keys that matter right now.

## Why it matters
Recognition-over-recall is what makes a TUI usable by a non-technical user on first run; they pick from what they see, not what they remember.

## Failure if missing
A blank prompt expecting a command the user has never been taught — the non-technical user bounces immediately.
