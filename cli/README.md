# @motherlabs/ada

Turn intent into governed Claude Code sessions.

## Install

```
npm install -g @motherlabs/ada
```

## Usage

```
ada compile "build a REST API for managing user accounts"
```

## Requirements

- Node.js >= 18
- Claude Code CLI (`npm install -g @anthropic-ai/claude-code`)
- `ANTHROPIC_API_KEY` environment variable
